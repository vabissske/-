
# Image Processor — Project 1 (Python + PyQt5)

Особенности проекта (соответствует "Проект 1" из задания):
- изменение яркости
- изменение контраста
- изменение размера изображения
- получение информации (ширина, высота, формат/режим)
- сохранение в PNG/JPEG
- загрузка через кнопку и drag & drop
- отображение исходного и обработанного изображения
- предпросмотр (preview)
- логирование операций (logs/app.log)
- история действий в configs/history.json
- простая отмена последнего действия (undo, хранит предыдущую версию)

Требования:
- Python 3.8+
- Pillow
- PyQt5

Как запустить:
1. Установите зависимости:
    pip install pillow pyqt5
2. Запустите приложение:
    python ui/main.py

Тесты:
    pytest
